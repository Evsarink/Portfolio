# EverInk Portfolio

Welcome to the EverInk Portfolio project! This repository contains the personal portfolio website for EverInk, a developer and creator. The design is inspired by cozy and artistic environments, aiming to create a warm and inviting experience for visitors.

## Project Structure

The project is organized as follows:

```
everink-portfolio
├── index.html           # Main HTML file for the portfolio
├── css                  # Folder containing all CSS files
│   ├── variables.css    # CSS custom properties for easy customization
│   ├── base.css         # Base styles for typography and layout
│   ├── layout.css       # Layout styles using grid and flexbox
│   └── components.css    # Styles for reusable components
├── js                   # Folder containing JavaScript files
│   └── main.js          # Main JavaScript file for interactions and animations
├── assets               # Folder for all asset files
│   ├── images           # Images used in the portfolio
│   ├── icons            # Icons used throughout the website
│   └── fonts            # Custom font files for typography
├── data                 # Folder for structured data
│   └── content.js       # Data for projects, skills, and personal information
└── README.md            # Project documentation
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser to view the portfolio.
3. Customize the content by editing the `data/content.js` file.
4. Modify styles by updating the CSS files in the `css` directory.

## Design Philosophy

The EverInk Portfolio is designed to evoke feelings of warmth and creativity. The layout and color palette are inspired by cozy cafés, vintage bookstores, and peaceful workspaces. The goal is to create an experience that feels handcrafted and inviting.

## Customization Guidelines

- All colors are defined as CSS custom properties in `css/variables.css`, making it easy to change the theme.
- Typography, spacing, and animations are organized into separate sections within the CSS files for clarity.
- The project structure allows for easy expansion with additional sections or projects in the future.

Thank you for visiting the EverInk Portfolio! We hope you enjoy your stay and find inspiration in the work showcased here.
